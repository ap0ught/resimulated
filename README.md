## Development

### 0: Required

- [node.js 6.13.1](https://nodejs.org/ja/) or higher
- ruby

### 1: Get Started

```sh
git clone git@github.com:gam0022/resimulated.git
cd resimulated

# init
npm install

# Start web server with hot-reload and UI for debug
npm run start

# Generate a dist\resimulated.html
npm run build
```
